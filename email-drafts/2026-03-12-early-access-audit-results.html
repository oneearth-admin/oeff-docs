Remove the passwords

Right now, supporters go through two password gates to get a ticket:

  1. Enter OEFF2026 to unlock each film page on Squarespace
  2. Enter getstheworm26 as a password to access each Eventbrite event page

Neither password is doing what we think it's doing. Here's why:

  • The email is already the gate. The film listing page is unlisted — no one can find it unless they click the link in the Mailchimp email. The individual film pages are only reachable from that listing. The Eventbrite events are unlisted too — not searchable, only reachable via direct link. So the access chain is already: email → listing → film page → Eventbrite. Each step is hidden from the public.
  • The Squarespace password adds friction, not security. A supporter who clicked through from the email has already proven they have access. Making them type a password just slows them down.
  • The Eventbrite password adds a second gate that isn't needed. The events are already unlisted and only reachable via direct link from the Squarespace film pages. The password just adds another step between a supporter and their ticket.

My recommendation: remove both. Take the password off the Squarespace film pages and drop the Eventbrite event password. Supporters should be able to go straight from the email to the film page to the ticket button in one smooth flow.

Update: I've already removed the Eventbrite passwords from all 14 events. The events are still private and unlisted — only reachable via direct link — but supporters no longer need to enter a code to see the event page. The Squarespace page passwords were also removed earlier.
