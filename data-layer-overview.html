The Four Layers

1

Connections

Link fields wire tables together. Each venue connects to its events, contacts, and intake form.

Venue → "2026 Events" → only this year's screenings
Venue → "Host Contacts" → contact people
Venue → "Host Intake" → form responses
2

Season Filter

The "2026 Events" link only includes events with Season = 2026. Old years are excluded at this level, before any data gets pulled.

28 events across 27 venues
(Columbia College has 2 screenings)
3

Summaries (LKP fields)

Rollup fields follow the connections and pull specific data into each venue's row. These are the "LKP" fields — short for lookup.

LKP Film Title → "Plastic People"
LKP Event Date → "2026-04-23"
LKP Event Time → "6:30P"
LKP OEFF Rep → "Kim"
4

Host Display

Formula fields format the raw data for hosts. If data is missing, they show a friendly fallback instead of blank cells.

Host - Film → "Plastic People"
Host - Screening Date → "Wednesday, April 23, 2026"
Host - Ticket Info → "Ticket link coming soon"
Host - Start Time → "6:30P"
