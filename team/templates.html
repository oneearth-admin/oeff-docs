Host Templates (H1–H6)

H1 Initial Host Inquiry
Subject: Welcome to OEFF 2026 — Next Steps Hi [First Name], Thanks for your interest in hosting a screening for One Earth Film Festival 2026! Here's what hosting involves: • Choose a film from our 2026 lineup • Screen it at your venue during April 22–26 • Host a post-screening conversation (we provide facilitator support) • Host fee: $600 (covers film license, materials, and support) Next step: Please complete our Host Interest Form: [INTEREST_FORM_URL] Once we receive your form, we'll follow up with film options and scheduling details. Questions? Reply to this email or check our Host FAQ: [FAQ_URL] Thank you, [Your Name] One Earth Film Festival
Use when: Someone emails asking about hosting for the first time.
H2 Question Answered in FAQ
Subject: RE: [Original Subject] Hi [First Name], Great question! This is covered in our Host FAQ. → [FAQ_URL#relevant-section] Let me know if that doesn't answer it or if you have follow-up questions. Best, [Your Name]
Use when: The answer is clearly in the FAQ. Always link to the specific section if possible.
H3 Packet Delivery
Subject: [Action by [CONFIRM_DATE]] Your OEFF 2026 Screening Packet — [Venue Name] Hi [First Name], Your screening packet is ready. Please confirm receipt by [CONFIRM_DATE]. Download link: [PACKET_URL] (Link expires: [EXPIRY_DATE]) Your packet includes: ✓ Film file: [Film Title] ([Format]) ✓ Captions: [Caption Type] ✓ Pre-show slides ✓ Pre-roll content (sponsor messages, accessibility announcement) ✓ Run of show document Next steps: 1. Download all files to your playback device 2. Test playback before your event (instructions in packet) 3. Reply to this email to confirm receipt Questions? Check our Host FAQ: [FAQ_URL] Or reply to this email. Thank you for hosting! [Your Name] One Earth Film Festival
Use when: Delivering the screening packet to a confirmed host.
H4 Technical Issue
Subject: RE: [Original Subject] Hi [First Name], Sorry you're running into trouble! Let's get this sorted. First, try these quick fixes: 1. Re-download the file (sometimes the download gets interrupted) 2. Try VLC Player (free at videolan.org) — it handles most formats better than default players 3. Make sure you're playing the main film file, not the caption file If still not working, please reply with: • What device/software are you using to play? • What error message (if any) do you see? • When is your screening? (so we can prioritize) We'll get you sorted quickly. [Your Name] OEFF Tech Support
Use when: Host reports playback issues. Get specifics before escalating.
H5 Webinar Reminder
Subject: [Reminder] OEFF Host Webinar — [DATE] at [TIME] Hi [First Name], Just a reminder: our next Host Webinar is [DATE] at [TIME]. Join link: [WEBINAR_URL] We'll cover: • [Topic 1] • [Topic 2] • Q&A Can't make it? No problem — recording will be shared within 24 hours. See you there! [Your Name]
Use when: Sending webinar reminders to hosts.
H6 Follow-up / Check-in
Subject: Checking in — [Venue Name] OEFF Screening Hi [First Name], Just checking in to see how things are going with your OEFF screening prep! Quick reminder of your details: • Film: [Film Title] • Date: [Screening Date] • Time: [Screening Time] Is there anything you need from us? Any questions that have come up? Let us know how we can help. Best, [Your Name]
Use when: Proactive check-in with hosts, especially approaching screening date.

Filmmaker Templates (F1–F4)

F1 File Request
Subject: [Action by [DEADLINE]] OEFF 2026 Film File Request — [Film Title] Hi [First Name], We need the screening file for "[Film Title]" by [DEADLINE]. Please upload to our secure transfer link: [UPLOAD_URL] File specs: • Format: H.264 MP4 or ProRes (preferred) • Resolution: 1920×1080 minimum • Audio: Stereo or 5.1 (labeled) • Include English captions if available (.srt sidecar or burned-in) If you need an extension or have questions about format, just reply to this email. Thank you, [Your Name] One Earth Film Festival films@oneearthfilmfest.org
Use when: Requesting screening file from filmmaker/distributor.
F2 File Received Confirmation
Subject: Received — [Film Title] Hi [First Name], We've received your file for "[Film Title]". Thank you! We'll run quality check and let you know if anything needs attention. Otherwise, you're all set on our end. Thanks for being part of OEFF 2026! [Your Name]
Use when: Confirming file receipt after QC passes.
F3 File Issue Found
Subject: [Action Needed] File Issue — [Film Title] Hi [First Name], We received your file for "[Film Title]" — thank you! However, we found an issue during quality check: Issue: [Describe specific issue] What we need: [Specific request — new file, captions, etc.] Please upload the corrected file to: [UPLOAD_URL] New deadline: [NEW_DEADLINE] Let us know if you have questions or need more time. [Your Name] OEFF Tech
Use when: QC found issue with received file.
F4 Follow-up (No Response)
Subject: [2nd Request] OEFF 2026 — [Film Title] File Needed Hi [First Name], Following up on our previous email — we still need the screening file for "[Film Title]". Current deadline: [DEADLINE] Upload link: [UPLOAD_URL] If there's an issue with the deadline or you need to discuss alternatives, please let us know as soon as possible. Thank you, [Your Name]
Use when: No response after first request. If no response after this, escalate.
