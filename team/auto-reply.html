Setup Instructions

Google Groups Auto-Reply Setup

  1. Go to groups.google.com
  2. Select the group (e.g., OEFF Tech)
  3. Click Group settings (gear icon) → Email options
  4. Find Auto-reply to incoming emails
  5. Toggle ON
  6. Paste the appropriate auto-reply text
  7. Save changes
  8. Send a test email to verify

Gmail Vacation Responder (Alternative)

If using Gmail directly instead of Groups:

  1. Go to Gmail → Settings → See all settings
  2. Scroll to Vacation responder
  3. Turn ON
  4. Set subject and message
  5. Check "Only send to people in my contacts" if desired
  6. Save

Setup Checklist

  • tech@oneearthfilmfest.org auto-reply configured
  • films@oneearthfilmfest.org auto-reply configured
  • All links tested and working
  • FAQ is live at linked URL
  • Test emails sent to each address to verify

When to Update Auto-Replies

  • Before festival week: Add/update hotline number
  • During festival week: Consider shorter response time commitment
  • After festival: Update to off-season message
  • If FAQ URL changes: Update all three auto-replies
Remember: Auto-replies only work if the FAQ is actually helpful. Keep FAQ updated based on questions that come in.
