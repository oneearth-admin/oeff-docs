What This Should Inform

This isn't about renegotiating. It's about three forward-looking decisions:

  • Delegation priority: Work that can be done by $15-40/hr hands (QC watch-throughs, packet assembly, discovery calls, data entry) should be delegated, not absorbed into this role. The micro-contract model makes that possible.
  • Scope discipline: New requests between now and festival should be evaluated against what's already in motion. If something is net-new, we should name it before starting it.
  • Succession investment: Building infrastructure that lets other people execute (Kim's onboarding, the SOW template, the team support system) is the highest-leverage use of this role. It converts expensive coordination time into reusable systems.
