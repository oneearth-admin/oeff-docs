# Deliverables Shipping — Stream 2

**Owner:** Kim
**Backup:** Garen
**Tools:** Mailmeteor, Google Drive, V7 Google Sheet, hosts@ Gmail
**Audience:** Practitioner
**Reading context:** Desk-doing — scan and execute when each deliverable window opens

---

## What this is

OEFF ships 5 categories of materials to hosts before their screenings: promotional materials, film files, slide decks, presenter guides, and t-shirts. Each has its own timing, delivery method, and tracking column. This SOP covers the shipping sequence for each deliverable and how to track what's been sent.

---

## The timeline

```
~3 weeks before screening (~Apr 1)
  → Promo materials (poster PDF, social graphics, event description)
  → Film file (MP4 via secure download, password-protected, per-host)

~2 weeks before screening (~Apr 8)
  → Presenter guide (tonight-runsheet.html)

~1 week before screening (~Apr 15)
  → Google Slides deck + trailers

1-2 weeks before screening
  → T-shirts (shipped or hand-delivered — need sizes first from Survey Form 1)
```

---

## Step by step

### 1. Confirm tracking columns exist in V7 Sheet — before first send

**What to do:**
1. Open the V7 Google Sheet (warm host tab)
2. Verify these columns exist: Film Delivered (Y/N), Promo Sent (Y/N), Slides Sent (Y/N), Presenter Guide Sent (Y/N), T-shirt Size, T-shirt Shipped (Y/N)
3. If any are missing, add them. Use Y/N format, not checkboxes (Mailmeteor reads text, not checkboxes)

**Decision rights:** You can add columns without approval. If you want to rename existing columns, check with Garen first — other tools may reference them.

---

### 2. Send screening packet — ~3 weeks before screening (~Apr 1)

**Template:** "Your screening packet is ready" (to be drafted — uses Mailmeteor)
**Merge fields:** {{First Name}}, {{Film Title}}, {{Download Link}}, {{Venue Name}}, {{Financial Password}}

**What to do:**
1. Confirm film files are uploaded and download links are ready (Garen provides these)
2. Confirm promo materials are in the shared Google Drive folder
3. Open Mailmeteor, select the screening packet template
4. Verify merge fields — especially {{Download Link}} which is per-host
5. Send test to yourself. Check: download link works, password works, promo links work
6. Send to full warm host list (or batch by screening date if staggered)
7. Mark "Film Delivered = Y" and "Promo Sent = Y" in V7 Sheet for each host

**Decision rights:** You own the send. If a host's download link isn't ready yet, skip them and follow up individually when it is.

**If something goes wrong:** If a host reports a broken download link, flag to Garen in Google Chat immediately — he manages the file hosting.

---

### 3. Send presenter guide — ~2 weeks before screening (~Apr 8)

**Template:** Part of the presenter guide email (to be drafted)

**What to do:**
1. Confirm the presenter guide is finalized and hosted (Garen handles the HTML build)
2. Send email with link to the presenter guide + link on host guide
3. Mark "Presenter Guide Sent = Y" in V7 Sheet

**Decision rights:** You own the send.

---

### 4. Send slides and trailers — ~1 week before screening (~Apr 15)

**Template:** "Slides and trailers for your screening" (to be drafted)
**Merge fields:** {{First Name}}, {{Film Title}}, {{Slides Link}}, {{Screening Date}}

**What to do:**
1. Confirm Google Slides deck is shared and accessible
2. Open Mailmeteor, select the slides/trailers template
3. Send test. Verify the Slides link opens correctly
4. Send to warm host list
5. Mark "Slides Sent = Y" in V7 Sheet

**Decision rights:** You own the send.

---

### 5. Track t-shirt shipments — 1-2 weeks before screening

**What to do:**
1. T-shirt sizes come from Survey Form 1 (Stream 4 — pre-event readiness form)
2. Sizes route to whoever handles ordering (check with Executive Director)
3. When shirts ship, mark "T-shirt Shipped = Y" in V7 Sheet
4. If a host didn't provide a size, follow up individually via hosts@ Gmail

**Decision rights:** You track and follow up. Ordering and budget route through Executive Director.

---

### 6. Follow up on non-opens — 48 hours after screening packet send

**What to do:**
1. Check Mailmeteor tracking for the screening packet email (Step 2)
2. If a host hasn't opened the email within 48 hours, send an individual follow-up from hosts@
3. Keep it short: "Just making sure you received your screening packet. Let us know if you need anything."

**Decision rights:** You own follow-ups. If a host is unresponsive after 2 attempts, flag to Garen.

## Escalation

- **Download link doesn't work** → Flag to Garen in Google Chat immediately. Don't try to fix the link yourself.
- **Host can't open the film file** → Likely a codec issue. Flag to Garen — he has troubleshooting steps.
- **Host hasn't responded after 2 follow-ups** → Flag to Garen. He or Executive Director may need to call directly.
- **T-shirt size wasn't collected** → Individual email from hosts@. Keep it light.
- **Budget question about shipping** → Route to Executive Director.
