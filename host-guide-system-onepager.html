How the Host Guide Works

OEFF 2026 — for the team

The Big Idea

Every host has a page on the Host Guide site with everything they need for their screening — date, time, film, parking, AV setup, contacts, and resources. Instead of emailing PDFs or sharing Google Sheets, hosts bookmark one URL that gets better over time as we add information.

The site is already live at hosts.oneearthfilmfest.org with general info, webinar recordings, and action cards. What we're adding now: venue-specific pages that show each host only their screening details.

How It Works

Hosts fill out
check-in form → Team adds
film + date + details → Garen runs
the generator → Venue pages
appear on site

The team never touches HTML or code. You update a spreadsheet (Airtable), the generator script turns that data into styled pages, and the site updates automatically. Garen controls the design; the team controls the content.

What Each Host Sees

The venue pages change automatically based on the time of year:

February
Confirmation status, assigned film title, general resources
March (now)
+ screening date/time, RSVP link, venue contacts, volunteer needs
April (festival month)
+ day-of timeline, screening packet, registration count, full run-of-show

Hosts don't see fields that aren't filled in yet. A venue page in February looks clean and simple; by April it has everything they need for the day of the screening.

Naming (So We're All Using the Same Words)

Term What it means
